# CLI Backend Projects

A collection of backend practice projects built as command-line (CLIs) applications in Python.  
Each task is isolated in its own folder with a separate virtual environment (excluded via `.gitignore`).

---

## 📁 Tasks Overview

| Task # | Project Name          | Description                          |
|--------|------------------------|--------------------------------------|
| 1      | Task Manager CLI       | Add, update, delete, and list tasks  |
| 2      | GitHub Activity Fetcher| Fetch user activity from GitHub      |
| 3      | Expense Tracker CLI    | Track daily expenses from terminal   |
| 4      | Number Guessing Game   | CLI number guessing game             |

---

## 🔧 How to Run a Task

1. Navigate to the task folder  
2. Activate its virtual environment  
3. Run the CLI script using the provided commands

For example (Linux/macOS):
```bash
cd task1
source venv/bin/activate
python main.py
